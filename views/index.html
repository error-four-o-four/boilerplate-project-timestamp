freeCodeCamp API Challenges

Timestamp Microservice

Example Usage:

timestamp/api/2015-12-25
timestamp/api/1451001600000

Example Output:

{ "unix" : 1451001600000, "utc" : "Fri, 25 Dec 2015 00:00:00 GMT" }

Request Header Parser Microservice

Example Usage:

/whoami/api/whoami

Example Output:

{
    "ipaddress" : "::ffff:159.20.14.100",
    "language" : "en-US,en;q=0.5",
    "software" : "Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"
}

URL Shortener Microservice

Use the form at /shortener/

Example Output:

{ original_url: "https://www.google.com", short_url: 1 }

Use the form at /exercise/
